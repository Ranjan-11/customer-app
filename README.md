# Customer App (Kafka Consumer)

## Description
Customer App is a Kafka consumer application that listens to hotel-related
Kafka topics and processes the incoming events for customer-related logic.

## Tech Stack
- Java
- Spring Boot
- Spring Kafka
- Apache Kafka

## Kafka Role
Consumer

## Topic
hotel-topic

## Consumer Group
customer-group

## How to Run
1. Start Zookeeper
2. Start Kafka
3. Run the application

mvn spring-boot:run
